# PWA Setup Guide

## 🎯 What We're Building

Three separate Progressive Web Apps:
1. **Rider PWA** - For booking rides
2. **Driver PWA** - For driver dashboard
3. **Admin PWA** - For managing drivers

All connect to the same backend: `http://localhost:8001/api`

## 🔧 Configuration Needed

Each PWA needs:
1. **vite.config.js** - PWA plugin configuration
2. **manifest.json** - App manifest for installation
3. **Service Worker** - Offline support
4. **API Configuration** - Connect to backend
5. **Routes** - App navigation
6. **Components** - UI components

## 🚀 Ports

- Rider PWA: `http://localhost:3001`
- Driver PWA: `http://localhost:3002`
- Admin PWA: `http://localhost:3003`
- Backend API: `http://localhost:8001`
