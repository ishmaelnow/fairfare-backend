# Admin PWA

Progressive Web App for admins to manage drivers with FairFare Transportation.

## 🚀 Features

- ✅ Installable (Add to Home Screen)
- ✅ Offline support
- ✅ Admin dashboard
- ✅ Approve/reject drivers
- ✅ Toggle driver availability
- ✅ Remove drivers
- ✅ Responsive design

## 📦 Installation

```bash
npm install
```

## 🏃 Run Development Server

```bash
npm run dev
```

Runs on: `http://localhost:3003`

## 🔗 Backend API

Connects to: `http://localhost:8001/api`

## 📋 Pages

- `/login` - Admin login
- `/dashboard` - Manage drivers

## 🔒 Authentication

Uses JWT tokens stored in localStorage.
Same authentication as web app.
