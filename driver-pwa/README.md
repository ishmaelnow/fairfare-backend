# Driver PWA

Progressive Web App for drivers to manage their rides with FairFare Transportation.

## 🚀 Features

- ✅ Installable (Add to Home Screen)
- ✅ Offline support
- ✅ Driver dashboard
- ✅ Apply to become driver
- ✅ View assigned rides
- ✅ Toggle availability
- ✅ Responsive design

## 📦 Installation

```bash
npm install
```

## 🏃 Run Development Server

```bash
npm run dev
```

Runs on: `http://localhost:3002`

## 🔗 Backend API

Connects to: `http://localhost:8001/api`

## 📋 Pages

- `/login` - Driver login
- `/register` - Driver registration
- `/driver-signup` - Apply to become driver
- `/dashboard` - Driver dashboard
- `/my-rides` - View assigned rides

## 🔒 Authentication

Uses JWT tokens stored in localStorage.
Same authentication as web app.
