# PWA Quick Start Guide

## 🎉 Rider PWA is Ready!

The Rider PWA is **complete and ready to use**!

## 🚀 Start Rider PWA

### Step 1: Install Dependencies
```bash
cd pwa-apps/rider-pwa
npm install
```

### Step 2: Start Backend (if not running)
```bash
cd ../../react-frontend-backend
.\venv\Scripts\Activate.ps1
python app.py
```

### Step 3: Start Rider PWA
```bash
cd ../../pwa-apps/rider-pwa
npm run dev
```

### Step 4: Open in Browser
- Opens automatically at: `http://localhost:3001`
- Or manually navigate to that URL

## ✅ Test the PWA

1. **Register** a new account
2. **Login** with your credentials
3. **Book a ride** with all details
4. **View dashboard** to see your rides
5. **Install as app** - Look for install button in browser

## 📱 Install as App

### Desktop (Chrome/Edge):
1. Look for install icon in address bar
2. Click "Install"
3. App opens in standalone window

### Mobile:
1. Open in browser
2. Menu → "Add to Home Screen"
3. App icon appears on home screen

## 🔗 Backend Connection

**Backend API:** `http://localhost:8001/api`

Make sure backend is running before using PWA!
